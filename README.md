# 法国诺曼底 2026 七月自驾

A data-driven FastAPI + Jinja2 site that visualizes a travel itinerary from
`app/data/trip.json`. Generated by the `trip-website` skill from a `trip.txt`.

## Run locally

```
uv sync
uv run uvicorn app.main:app --reload
```

Open http://127.0.0.1:8000 — the start page is the trip overview.

## Deploy

Railway-ready via `Procfile` (`uvicorn app.main:app`).
